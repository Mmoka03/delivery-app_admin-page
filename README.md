# 과제용 배달앱 관리자 페이지

## 무슨 프로젝트인가요?

과제용으로 개발중인 배달앱 관리자 페이지입니다. <br />
회원, 쿠폰, 음식점을 중심으로 관리할 수 있도록 설계되었습니다.
<br />

## 어디까지 진행되었나요?

간단한 뷰와 DB는 완성되었습니다. <br />
메뉴에 해당하는 페이지를 완성시키면 끝입니다.
